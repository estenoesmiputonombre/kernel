# Multiplexing

## Definition

Method by which multiple **analog** or **digital** signals are _combined_ into one signal over shared medium.

## Schema

```
Data* = Multiple low data rate signals
$     = single high data rate link

Data*       Multiplexing  Demultiplexing
--------> |\            /| --------->
--------> | \    $     / | --------->
--------> |  | -----> |  | --------->
--------> | /          \ | --------->
--------> |/            \| --------->
```
